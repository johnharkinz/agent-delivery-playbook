# Agent Delivery Playbook

This repo contains reusable workflow assets for agent-assisted software development.

It defines:

- skills: lightweight ways of working with AI tools
- templates: starter files for new project repos
- examples: reference material from proven project usage

Projects remain self-contained.

Templates should be copied into project repos and adapted. This repo is not a runtime dependency.

## Intended Workflow

```text
idea / conversation
  |
  v
clear product or technical plan
  |
  v
small GitHub issues
  |
  v
Codex implementation
  |
  v
PR review
  |
  v
useful project memory
```

## Current Principle

Start small.

Do not build a large skill library upfront. Do not create a separate project-management system. Use this repo to support practical delivery, not to replace judgement.

## First Milestone

The first milestone is to create the minimum reusable playbook assets, then trial them on Shareable.

Do not expand this repo until the first real Shareable trial has been reviewed.
